About smartX...

A few words

smartX project is a small web application, allowing to check status of hard disks connected to a VMWare ESXi hypervisor.

It does not directly fetch information from ESXi, it rather relies on REST services as a client; compatible services are provided by pi-control project, currently.

Why this project ? Simply because getting S.M.A.R.T such disk results in a simple and modern way did not exist!
